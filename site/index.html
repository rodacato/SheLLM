SheLLM

Your LLM services — unified as a REST API

API Docs GitHub

Most LLM gateways assume you're paying per-token via API key. SheLLM's primary use case is different: you already have CLI subscriptions (Claude Max, Gemini AI Plus, OpenAI Enterprise) and want to expose them as a regular HTTP API to your apps — without burning API quota.

Quick Start

# Clone and install
git clone https://github.com/rodacato/SheLLM.git && cd SheLLM
npm install && npm link

# Configure and start
cp .env.example .env
shellm start

# Send a request
curl http://localhost:6100/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"model":"claude","messages":[{"role":"user","content":"Hello"}]}'
